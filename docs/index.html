/tokens

Mapped to AWS Cognito's /oauth2/token endpoint, this request allows users to retrieve access tokens for authenticating and authorizing Mainstay Fax client applications.

Header Parameters

Authorization string REQUIRED

Contains the credentials used to authenticate the client making the request for authorization purposes.

Body Parameters

grant_type string

Specifies the authorization grant type requested by the OAuth2 token endpoint.

scope string

Defines the access scope and resources the issued token can use.

Response

200 Object
429 Object

Too Many Requests.

500 Object

Internal error.

Response Attributes

access_token string

Bearer token used for authenticated Mainstay Fax requests.

expires_in number

Token lifetime, returned in seconds.

token_type string

Token type returned by the authorization service.
