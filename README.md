# Aizan Mainstay Fax API Docs

Static GitHub Pages documentation for the Aizan Mainstay Fax API.

## Local Preview

From this folder, run:

```bash
python3 -m http.server 8021 --directory docs
```

Then open:

```text
http://localhost:8021/
```

## GitHub Pages

Recommended repository: `Aizan-Technologies-Inc/mainstay-fax-api`

Publish from:

- Branch: `main`
- Folder: `/docs`
